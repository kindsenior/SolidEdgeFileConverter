* 使い方

あらかじめSolidEdgeを開いておく.
真ん中の大きな白い四角にdftファイルをドラッグ&ドロップするか,追加ボタンで追加する.
auto load settingの四角にxmlファイルをドラッグ&ドロップするか,追加ボタンで追加する.

dest dirの右の四角に出力先フォルダをドラッグ&ドロップするか,選択ボタンで選択する.

あとはto PDF and DXFなどのボタンを押すと,変換がはじまる.


* tips

処理する前にSolidEdgeを最小化しておくと,画面がチラチラしないので早くなる.
